# Q1. Java에서 람다식을 사용할 때 클래스는 안되나?
- Java에서 람다식은 함수형 인터페이스만을 대상으로 설계되어 있다.
- 클래스(추상 클래스 포함)는 상태, 생성자, 여러 메서드를 가질 수 있어 람다의 대상이 되지 않는다.
- 따라서 람다는 단순히 메서드 수가 아니라 “인터페이스 여부”에 의해 결정된다.

# Q2. Java와 Python 람다식의 차이는?
- Python은 함수가 객체이기에 바로 람다식 사용이 가능하다.
- Java는 함수가 객체가 아니기에 함수형 인터페이스로 람다를 활용해야 한다.
