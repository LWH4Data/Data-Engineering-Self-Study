<h1>1. 컬렉션 프레임워크</h1>
<ul>
  <li>
    자바는 널리 알려져 있는 <strong>자료구조(Data Structure)</strong>를 사용할 수 있도록 <strong>java.util 패키지</sstrong>에 포함시켜 두었으며 이들을 총칭하여 <strong>컬렉션 프레임워크(Collection Framework)</strong>라 부른다.
  </li>
  <li>
    컬렉션 프레임워크에서는 몇 가지 인터페이스를 제공하는데 주요 인터페이스로는 List, Set, Map이 있으며 각각 컬력션 객체를 제공한다. (p642 참고).
  </li>
</ul>

<br><br>

<h1>2. List 컬렉션</h1>
<ul>
  <li>
    List 컬렉션은 객체를 <strong>인덱스</strong>로 관리하며 객체를 저장하면 <strong>인덱스가 부여</strong>되고 <strong>인덱스로 객체를 검색, 삭제할 수 있는 기능</strong>을 제공한다.
  </li>
  <li>
    List 컬렉션에서는 ArrayList, Vector, LinkedList 등을 제공하며 자세한 인터페이스 메소드들은 p643을 참고.
  </li>
</ul>
