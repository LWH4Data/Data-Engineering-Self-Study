<h1>1. 디버깅 가이드</h1>
<ul>
  <li>
    디버깅 기능은 객체에 어떤 값이 들어가는지, 메서드를 실행하면 어떤 값을 반환하는지 등을 확인할 때 사용한다.
  </li>
  <li>
    멈추고 싶은 부분을 <strong>브레이크 포인트</strong>로 설정하고 벌레 모양의 실행 버튼을 눌러가며 확인한다.
  </li>
    <ul>
      <li>
        <strong>포스트맨</strong>으로 요청을 보낸 후 서버에서 확인을 하여도 브레이크 포인트에 멈추어 있고 결과를 반환한다.
      </li>
    </ul>
</ul>
