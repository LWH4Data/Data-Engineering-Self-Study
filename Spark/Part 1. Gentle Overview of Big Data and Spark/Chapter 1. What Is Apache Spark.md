# Chapter 1. What Is Apache Spark
<ul>
  <li>
    Apache Spark는 <a href="#computer-clusters">computer clusters</a>에서 데이터 병렬 처리를 위해 사용되는 통합된 컴퓨팅 엔진이자 라이브러리들의 모음.

      <li>단일 labtop부터 수천개의 서버를 포함하는 클러스터에도 적용 가능.</li>
